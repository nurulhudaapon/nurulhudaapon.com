## Developing
Requirements: `zig` (0.17.0-dev.1398)

Start dev server (http://localhost:3000):
```bash
zig build dev
```

Build static site (output in `dist/`):

```bash
zig build zx -- export
```
Built with

- Ziex: https://ziex.dev
- Zig: https://ziglang.org
